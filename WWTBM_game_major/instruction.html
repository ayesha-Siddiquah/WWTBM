INSTRUCTIONS

WHO WANTS TO BE A MILLIONAIRE is a single-player game. The user has to answer 6 questions correctly to win $1,000,000 out of the 8 questions. Each question is given 30 seconds for the player to answer. If in 8 questions, the user answers 3 questions incorrectly, then the game is over. Also, the game is over when you fail to answer any question within 30 seconds. There are three levels in this game. The user can choose any level. If the player gets a score of 6 . Then he is the winner.

LEVELS

Level I - Easy

In this level, the user has the chance to use 2 lifelines: 50-50 and STOP-TIMER.
The player wins $1,000,000.

Level II - Intermediate

In this level, the user has the chance to use only one lifeline: 50-50.
The player wins $5,000,000.

Level III - Challenging

In this level, the user has no chance to use any lifeline.
The player wins $10,000,000.

50-50

In this lifeline, four characters of the answer are revealed, so that the user can take help of this and answer the question. If the question is correct, the score is incremented by one and the life-line used counter is also incremented by one. Once used this lifeline cannot be used again. If the user successfully answers six questions, he goes home with $1,000,000.

STOP -TIMER

In this level, the user can stop the timer and give the answer without any hurry. If the question is correct, the score is incremented by one and the life-line used counter is also incremented by one. Once used this lifeline cannot be used again. If the user successfully answers six questions, he goes home with $1,000,000.
